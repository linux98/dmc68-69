# DMC Interactive Executive Dashboard (2568 - 2569)

ระบบรายงานผลเชิงบริหารวิเคราะห์สถิตินักเรียน ห้องเรียน และอัตราส่วนระหว่างปีการศึกษา 2568 และ 2569 สังกัด สพม.สกลนคร (จำนวน 45 โรงเรียน) 
ได้รับการออกแบบใหม่โดยอิงตามรูปแบบ **UX/UI Light Mode Minimal จากระบบผู้ตรวจราชการ (Mission BOSS)** และดึงฟอนต์ **Google Sans** ร่วมกับ **Sarabun** มาใช้งานโดยตรงตามรูปแบบมาตรฐาน

*ระบบได้รับการอัปเดตเพื่อแก้ปัญหากลุ่มเมนูด้านซ้ายไม่ได้ใช้งาน (Static menu) ให้กลายมาเป็นแถบนำทางและควบคุมระบบแบบมีปฏิสัมพันธ์ (Interactive Navigation & Settings) ทำให้ผู้บริหารสามารถใช้งานได้อย่างเกิดประโยชน์สูงสุดในที่ประชุม*

---

## สรุปการปรับปรุงเมนูแบบมีปฏิสัมพันธ์ (Sidebar & Settings Specs)
1. **การควบคุมการนำทางในหน้าจอ (Scroll-to & Glow Highlights)**:
   * **เมนูแดชบอร์ด**: คลิกแล้วเลื่อนกลับไปด้านบนสุด (ภาพรวม Dashboard) เสมอ
   * **เมนูสถิตินักเรียน**: เลื่อนจออย่างลื่นไหล (Smooth Scroll) ไปยังกราฟเปรียบเทียบนักเรียนทันที พร้อมทั้งใส่เอฟเฟกต์กะพริบเรืองแสงสีฟ้า (Glow Highlight) รอบแผงกราฟ 1.5 วินาทีเพื่อเป็นจุดนำสายตา
   * **เมนูสถิติห้องเรียน**: เลื่อนหน้าจอไปยังส่วนแสดงสถิติห้องเรียนและตัววิเคราะห์ความหนาแน่น พร้อมทั้งเรืองแสงนำสายตาที่การ์ดสถิติห้องเรียนรวมและการ์ดสถิติความหนาแน่นเฉลี่ย
   * **เมนูวิเคราะห์รายเขต**: เลื่อนหน้าจอไปยังตารางเปรียบเทียบข้อมูลรายโรงเรียนทั้งหมดด้านล่างทันที
2. **การตั้งค่าระบบผ่านกล่องควบคุม (Settings Modal)**:
   * **เมนูตั้งค่าระบบ**: คลิกแล้วจะแสดงผล **กล่องตั้งค่า (Modal Overlay)** แบบลอยขึ้นมาพร้อมเอฟเฟกต์ Blur ฉากหลังอย่างเรียบหรู
   * **การปรับสีเน้นย้ำสด (Live Accent Colors)**: สามารถเลือกสีระบบใหม่ได้ 5 สีตามต้องการ ได้แก่ Blue (น้ำเงิน Google), Green (เขียว Google), Red (แดง Google), Orange (ส้ม Google) และ Purple (ม่วงหลวง)
     * เมื่อคลิกเลือกสี: ธีมสีเน้นย้ำของปุ่ม ลิงก์ ไฮไลท์ แถบเมนูด้านซ้าย รวมถึง **เส้นกราฟหลักและเฉดสีถมกราเดียนต์ใต้กราฟจะเปลี่ยนโทนสีตามทันทีแบบ Real-time**
   * **การเปิด-ปิดองค์ประกอบ**:
     * **แสดงเส้นตารางเบื้องหลังกราฟ (Gridlines)**: ติ๊กเปิด-ปิดเพื่อแสดงหรือลบเส้นตารางฉากหลังบนกราฟ Chart.js ได้ทันที
     * **แสดงกรอบแจ้งเตือนสถิติวิกฤต (Alert Panel)**: ติ๊กเปิด-ปิดเพื่อซ่อนหรือแสดงแผงกรอบแจ้งเตือนสีแดงในหน้าแดชบอร์ดได้ทันที

## โครงสร้างไฟล์ในโปรเจกต์
* **`index.html`** - หน้าแดชบอร์ดระบบ (มี Sidebar, คอนโทรลฟิลเตอร์, การ์ด KPI, แผงกราฟ/Progress bars, ตาราง และกล่องตั้งค่าระบบลอยตัว Settings Modal)
* **`style.css`** - สไตล์ชีตหลักสีขาว-เทาจาง-ฟ้า ตามต้นแบบ Mission BOSS และดึง Google Sans มาใช้ รวมถึงสไตล์ชีตสำหรับกล่องตั้งค่าระบบและอนิเมชั่นเรืองแสงนำสายตา
* **`app.js`** - ตรรกะประมวลผลและการโต้ตอบ (Interactive) เช่น การกรองข้อมูล, การนำทางด้วยเมนู, และตรรกะระบบตั้งค่าปรับเปลี่ยนโทนสีแดชบอร์ด/กราฟแบบไดนามิก
* **`data.js`** - ไฟล์ฐานข้อมูล JSON ที่ถูกรวมและจัดระเบียบข้อมูลจากปี 2568 และ 2569-1
